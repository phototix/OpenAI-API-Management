Important Information

  • Get an organization Admin Key at OpenAI Admin Keys.
  • This app sums usage via /v1/organization/costs using an Admin Key. It queries a date range (start_time inclusive, end_time exclusive); for long ranges it may chunk by 7 days. Cards show total Used only.
  • Keys are stored locally in your browser (localStorage) and are only sent to OpenAI's API from your device. You can remove them anytime using ‚ÄúClear All Keys‚Äù.
  • Browser requests may be blocked by CORS. If needed, run a local proxy and set localStorage.setItem('openai_cors_proxy', 'http://localhost:8787') in the console.
  • How to use: (1) Enter an account name and Admin Key. (2) Pick a range via the calendar button. (3) Click ‚ÄúRefresh All‚Äù. You can add multiple accounts/keys.
  • Where to get a key: Create a key in the OpenAI Platform (API Keys) using a user with organization admin privileges, so it can access organization costs.
